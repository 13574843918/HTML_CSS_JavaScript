⚡ HBuilderX 使用技巧 · 本节专属 查找替换 & 选择器调试
🔍
Ctrl + H

打开"查找替换"对话框。当你要把页面中所有 .old-class 更名为 .new-class 时，一次性全部替换，避免遗漏。

🧩
ul>li*3 按 Tab

Emmet 语法：输入 ul>li*3 按 Tab，一键生成含 3 个 <li> 的无序列表，快速构建搜索结果结构。

🎨
颜色选择器

在 CSS 中输入颜色值时，HBuilderX 会弹出可视化颜色选择器。直接点选颜色，无需记忆十六进制代码，修改搜索页面高亮色非常直观。

📌
div.header>h2+p 按 Tab

Emmet 组合写法：生成一个 class 为 header 的 div，内含 h2 和 p 标签。构建搜索页面 HTML 结构时，用这种方式可以几秒内完成骨架搭建。

🐞
右键 → 检查元素

在浏览器预览中右键任意元素点"检查"，可以看到该元素实际生效的 CSS 规则及其来源选择器，是理解层叠性和优先级的最佳工具。

📂
Ctrl + Shift + F（全局查找）

在整个项目中搜索某个选择器（如 .special），快速定位它在哪些 HTML 和 CSS 文件中被使用，理清选择器的使用范围。

复合选择器权重的叠加

实训目标：理解权重叠加计算规则

关键知识点
权重计算

类选择器权重(10) > 标签选择器权重(1)

不进位规则

权重不会进位，无数标签叠加也不大于类选择器

就近原则

权重相同时，后定义的样式生效
