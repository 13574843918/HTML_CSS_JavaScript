🗂️
文件 → 新建 → 项目

选择"普通项目"类型，类型选 Web，自动生成 index.html 入口文件，省去手动创建的步骤。

📄
输入 html 按 Tab

在空白 .html 文件中输入 html 再按 Tab，瞬间生成完整的 HTML5 文档结构，含 charset、viewport 等必要 meta。

🔗
输入 link:css 按 Tab

快速生成 <link rel="stylesheet" href="style.css">，立即链接外部样式表，无需记忆完整写法。

🖥️
Ctrl + R

保存后按 Ctrl+R 在浏览器中实时预览，修改 CSS 立刻看到效果变化，无需手动刷新浏览器。

✏️
输入 .className 按 Tab

在 CSS 文件中输入 .red 再按 Tab，自动补全为 .red { } 并将光标置于花括号内，高效书写类选择器。

💡
Ctrl + /

选中一行或多行代码后按 Ctrl+/ 可快速切换注释（/* */），调试 CSS 时非常实用。
