# HTML+CSS 网页设计实训平台

## 产品概述

本平台是一个以实训任务驱动的HTML+CSS网页设计学习网站，帮助学习者通过实际操作掌握前端开发核心技能。学习者完成实训后能在HBuilderX中独立开发，熟练掌握开发工具。

## 核心功能

### 1. 实训任务导航

- **左侧导航栏**：清晰展示三个实训任务，方便快速切换
- **任务卡片**：每个任务显示编号、标题和简介
- **进度指示**：当前任务高亮显示，带有视觉指示器

### 2. 任务一：文字Logo

**学习主题**：CSS选择器入门

包含以下示例：
- Example 01：内嵌式引入CSS样式表
- Example 02：外链式引入CSS样式表
- Example 03：类选择器的使用
- Example 04：ID选择器的使用
- Example 05：文字Logo制作
- 扩展任务：CSS3炫彩文字Logo

**知识点**：
- `<style>` 标签的使用
- `<link>` 标签引入外部样式
- 类选择器 `.` 与ID选择器 `#`
- 选择器优先级规则

### 3. 任务二：美食专题栏目

**学习主题**：CSS字体与文本属性

包含以下示例：
- Example 06：字体属性综合设置
- Example 07：服务器字体引入
- Example 08：字间距与词间距
- Example 09：行高设置
- Example 10-16：文本装饰、颜色、对齐等

**知识点**：
- `font` 属性简写语法
- `@font-face` 自定义字体
- `letter-spacing` 与 `word-spacing`
- `line-height` 三种单位设置
- `text-decoration`、`text-align` 等文本属性

### 4. 任务三：搜索页面

**学习主题**：复合选择器与CSS三大特性

包含以下示例：
- Example 17：标签指定式选择器
- Example 18：后代选择器
- Example 19：并集选择器
- Example 20：CSS层叠性
- Example 21：复合选择器权重叠加
- Example 22：搜索页面案例实现

**知识点**：
- 复合选择器的定义与使用
- CSS层叠性、继承性、优先级
- 选择器权重计算规则

## 界面设计特点

### 视觉风格
- **现代深色主题**：以深蓝黑为基调，橙色为强调色
- **渐变背景**：营造层次感和科技感
- **卡片式布局**：清晰分隔不同内容模块

### 交互体验
- **折叠展开**：示例卡片可展开查看详细内容
- **一键复制**：代码块支持一键复制功能
- **代码高亮**：自动高亮HTML/CSS语法
- **知识点网格**：结构化展示关键知识点
- **快捷键支持**：数字键1-3快速切换任务

### 响应式设计
- 桌面端：左侧固定导航 + 右侧内容区
- 平板端：优化布局和字体大小
- 移动端：导航可折叠，适应小屏

## 扩展性设计

### 添加新实训任务

1. 在HTML中添加新的 `<section class="task-section" id="taskN">`
2. 在导航中添加对应的 `<li class="nav-item" data-task="taskN">`
3. 按照现有结构添加示例卡片

### 添加新示例

1. 复制现有的 `.example-card` 结构
2. 更新 `data-example` 属性
3. 填写内容：标题、步骤、代码、知识点

## 技术栈

- **前端框架**：原生HTML5 + CSS3 + JavaScript
- **字体**：Google Fonts (Noto Sans SC, JetBrains Mono, ZCOOL KuaiLe)
- **图标**：内联SVG
- **代码高亮**：自定义JavaScript实现

## 使用指南

### 学习者使用流程

1. 打开网站，查看左侧导航的任务列表
2. 点击任务进入对应学习内容
3. 点击示例卡片展开查看详细步骤
4. 阅读代码和知识点讲解
5. 在HBuilderX中创建项目，输入代码
6. 保存并在浏览器中预览效果
7. 截图保存学习成果

### 开发工具要求

- **HBuilderX** 3.0及以上版本
- **浏览器**：Chrome、Firefox或Edge
