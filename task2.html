🖱️
Alt + 点击

在代码中按住 Alt 再点击多处，可同时添加多个光标。一次修改所有段落的 font-size，效率翻倍。

🎯
Ctrl + D

选中一段文字后按 Ctrl+D，依次选中下一个相同的词。批量修改多处相同的属性值（如颜色值 purple）时非常方便。

🔧
Ctrl + Shift + F

一键格式化代码，自动整理缩进和空格。写完字体样式后用它让代码立刻变整洁，特别适合 CSS 属性较多时使用。

💬
输入属性名按 Tab

在 CSS 中输入 fs 按 Tab，自动补全 font-size: ;；输入 lh 补全 line-height:，记住常见缩写可大幅提速。

📋
Ctrl + Shift + R

复制当前行到下一行。写多个具有相似结构的 CSS 规则时，先写一条再用此快捷键复制，比重新输入快很多。

🔍
F12 开发者工具

在浏览器预览页面后按 F12 打开开发者工具，在"元素"面板可实时修改 font-family、color 等属性并立即预览效果。
