4-13

块元素和行内元素

实训目标：理解块元素与行内元素的区别及各自特点

关键知识点
块元素

独占一行，可设宽高

行内元素

不独占一行，不可设宽高

常见块元素

div/h1-h6/p/ul/ol/li

常见行内元素

strong/em/del/span

动手练练
  • 尝试给 strong 添加 width: 200px;，观察宽度设置是否生效
  • 尝试给 h2 添加 display: inline;，观察块元素变行内
  • 观察块元素（h2、p）是否独占一行
挑战练习
  • 给 p 添加 display: inline-block;，观察可设宽高的行内块效果
  • 给行内元素添加 padding: 10px;，观察内边距对周围的影响

span标签用法

实训目标：掌握span标签作为行内容器的用法，与div的区别

关键知识点
span

行内元素，作为文本容器

vs div

span是行内，div是块级

用途

配合class定义特殊文本样式

动手练练
  • 将 span.red 的 color: red 改为 background: yellow，为文字添加背景色
  • 添加新的class：span.bold { font-weight: bold; } 并应用
  • 将 span 改为 div，观察布局变化
挑战练习
  • 给 span 添加 display: inline-block; 和 padding: 5px;
  • 添加 border-radius 创建标签样式效果
4-15

display属性转换

实训目标：掌握display属性的四种值：inline、block、inline-block、none

关键知识点
inline

转行内元素，不独占一行

block

转块元素，独占一行

inline-block

转行内块，可设宽高但不独占一行

none

隐藏元素，不占空间

动手练练
  • 将 .inline 改为 display: block;，观察div独占一行
  • 删除 .none 的 display: none;，观察隐藏元素显现
  • 将 span 添加 display: block;，观察行内变块级
挑战练习
  • 尝试给 inline-block 元素添加 vertical-align: middle;
  • 使用 visibility: hidden; 对比 display: none; 的区别

图标导航栏（案例实现）

实训目标：综合运用display:inline-block和背景图片制作图标导航栏

inline-block

使span可设宽高又不独占一行

nth-child

为不同导航项设置不同背景图

为导航栏添加底部边框线

使导航图标从左侧留白开始

  • 为导航项添加悬停效果：:hover { background-color: #ddd; }
  • 修改底部边框颜色为其他颜色
